In-Page Slider Floating Display

In-Page

Unidade nativa embutida no fluxo do artigo.

Características:

1. Visibilidade:

Permanece visível na tela mesmo quando o usuário rola para cima ou para baixo na página, garantindo maior visibilidade em comparação com anúncios estáticos.

2. Posicionamento:

Aparecerá em várias posições na página web, como nos cantos ou no centro. A posição é fixa por padrão até que o usuário comece a rolar, significando que segue o usuário enquanto ele rola. Você verá um exemplo de demonstração que aparecerá entre o texto.

3. Tamanho e Dimensões:

Esses anúncios são menores em tamanho em comparação com outros formatos de anúncios de vídeo para evitar que se tornem muito intrusivos. O tamanho é ajustável, permitindo que os usuários minimizem ou maximizem a janela de vídeo.

4. Conteúdo:

Esses anúncios contêm conteúdo de vídeo, que pode variar desde anúncios de produtos até vídeos promocionais, com uma chamada para ação (CTA) clara para incentivar o engajamento do usuário.

Interação do Usuário:

1. Fechável:

Os usuários têm a opção de fechar o anúncio se desejarem, o que ajuda a manter uma experiência amigável ao usuário.

2. Clicável:

Os anúncios são clicáveis, redirecionando os usuários para uma página de destino específica quando clicados.

3. Opção de Silenciar/Ativar Som:

Esses anúncios vêm com uma opção para que os usuários silenciem ou ativem o áudio.

4. Controles:

Os usuários podem ter controle sobre a reprodução do vídeo, com opções para reproduzir, pausar ou rebobinar o vídeo.

Análise e Rastreamento:

1. Rastreamento de Impressões:

Os anunciantes podem rastrear o número de vezes que o anúncio foi visualizado (impressões).

2. Rastreamento de Interações:

Os anunciantes também podem rastrear várias interações do usuário com o anúncio, como cliques, visualizações de vídeo e fechamentos para avaliar o desempenho do anúncio.

3. Métricas de Visibilidade:

Esses anúncios vêm com métricas de visibilidade para ajudar os anunciantes a entender quanto de seu anúncio foi visualizado e por quanto tempo.

Aspectos Técnicos:

1. Design Responsivo:

Tem um design responsivo para se adaptar a vários tamanhos de tela e dispositivos.

2. Tempo de Carregamento:

Para prevenir uma experiência negativa do usuário, esses anúncios são otimizados para ter tempo de carregamento mínimo.

3. Codificação:

Foi codificado usando HTML5, CSS e JavaScript para permitir integração suave com páginas web e compatibilidade com navegadores modernos.

Conformidade e Melhores Práticas:

1. Conformidade de Privacidade:

Cumpre com regulamentos e diretrizes de privacidade, que podem incluir dar aos usuários a opção de optar por não coletar dados.

2. Experiência do Usuário:

Os anunciantes equilibram capturar a atenção do usuário com manter uma experiência positiva do usuário, evitando formatos de anúncios muito intrusivos ou disruptivos.
